# Анализ данных киноиндустрии

Проведено исследовательское исследование данных о фильмах 2018-2019 годов. В ходе анализа определены лидеры по профессиональным оценкам критиков (Metascore), пользовательским рейтингам (IMDb), количеству зрительских голосов и кассовым сборам.

## Датасет содержит:

- Unnamed: 0 - индекс;
- movie_name - название фильма;
- genre - жанр фильма;
- year - год выхода в прокат;
- timeMin - длительность фильма;
- imdb - оценка IMBb;
- metascore - оценка Metacritic;
- votes - количество голосов;
- us_grossMillions - кассовые сборы в США.


## Стек:
- numpy
- pandas
- seaborn
- matplotlib
